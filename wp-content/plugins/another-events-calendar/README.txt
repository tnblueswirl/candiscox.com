== Description ==

Plan and supervise your events with bliss using Another Events Calendar which comprises of Calendar, Venue Management, Handle Organizers, Google Map Integration and plenty more!!! 

[Demo](http://demo.yendif.com/wordpress/) | [Documentation](http://yendif.com/wordpress/another-events-calendar/getting-started.html) | [Support](http://yendif.com/forum/another-events-calendar-for-wordpress.html)

Make your events fame using Another Events calendar with its pleasing features.

= Features (Back-End) =

* Limitless Event Categories
* Limitless Event Tags
* Manage Event Venues
* Manage Event Organisers
* Powerful & Simplified Event form
* Customize front-end display
* Google Maps Integration
* Built-in Social share system
* Clean code. No custom tables used. Uses "Custom Post Types", "Custom Taxonomies" and "Settings API" to manage the data. Strictly follows all the WordPress standards.
* [+] lot more

= Features (Front-End) =

* Allow users to add their Events, Venues & Organizers through front-end.
* Showcase your Events in 4 nice layouts,
 * Calendar (Monthly, Weekly & Daily)
 * Table
 * Grid
 * Blog
* Show Events by Category
* Show Events by Location
* Show Events by Organiser
* Show Upcoming Events only, Past Events only or Both
* Search engine friendly URLs
* 100% Responsive. Uses Bootstrap 3 for design( Has option to enable/disable the loading of Bootstrap from the plugin back-end)

Note: If your theme already uses Bootstrap, then you're lucky and this plugin will serve you greatly to showcase your events in your website nicely. If your theme don't use Bootstrap, then the plugin uses namespaced Bootstrap version to avoid conflicts with your template styles. So, in neither case Another Events Calendar helps you to showcase your events nicely in your website.

= Widgets =

* Widget to search events in your website
 * Search by event title, description
 * Search by event category
 * Search by single date
 * Search by a date range
* Widget to show upcoming events
* Widget to show events using a mini calendar

== Installation ==

1. Download the plugin.
2. From the WordPress Admin Panel, click on Plugins => Add New.
3. Click on Upload, so you can directly upload your plugin zip file.
4. Use the browse button to select the plugin zip file that was downloaded, and then click on Install Now.
5. Once installed, click "Activate".

That's it! Visit the official [Documentation](http://yendif.com/wordpress/another-events-calendar/getting-started.html) page for further setup.

== Frequently Asked Questions ==

= Where can I find complete documentation? =
Full searchable docs can be found at (http://yendif.com/wordpress/another-events-calendar/getting-started.html)

= What themes work with Another Events Calendar? =
Any properly written theme will work with Another Events Calendar.

= Getting a 404 error? =
To get rid of the 404 error when viewing a download, you need to resave your permalink structure. Go to Settings > Permalinks and click "Save Changes".
